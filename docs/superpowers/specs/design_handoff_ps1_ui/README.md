# Handoff: Idea Brewing — PS1風UIリデザイン

## Overview
アイデア醸造ツール「Idea Brewing」(Next.js アプリ、FUJIvillage/idea-brewing)の全画面を、初代プレイステーション時代のゲームUI(ローポリ・ディザ・CRT・メモリーカード画面風)として再デザインしたもの。ビール醸造メタファー(タンク・泡・アンバー色)は維持しつつ、PS1風に再解釈している。

対象画面: 起動画面 / ホーム(醸造タンク一覧) / 新しい仕込み / ワークベンチ7タブ(原料・ブリューシート・グリル・レシピ・タップ・熟成・Pub) / リーダーボード / 設定。

## About the Design Files
このバンドルの `Idea Brewing PS1.dc.html` と `ps1-tank.js` は **HTMLで作られたデザインリファレンス** です。見た目と挙動を示すプロトタイプであり、そのまま本番コードにコピーするものではありません。タスクは、このデザインを **対象コードベース(Next.js App Router + React + TypeScript + Tailwind)の既存パターンで再実装すること** です。既存のページ構成(`src/app/`)・コンポーネント構成(`src/components/`)・状態管理はそのまま使い、見た目のレイヤーを置き換えてください。

例外: `ps1-tank.js` の3D描画ロジック(canvas)は数式・定数ごとほぼそのまま移植可能です(Reactコンポーネントでラップ)。

## Fidelity
**High-fidelity (hifi)**。色・タイポグラフィ・余白・ボーダー・エフェクトはすべて最終値。ピクセルパーフェクトに再現してください。

- **プライマリボタン**: `background:#d98a12; color:#140a02; border:2px solid; border-color:#ffd88a #7a4a06 #7a4a06 #ffd88a`(明暗ベベル)+ `box-shadow:3〜4px 3〜4px 0 #000`。hover: `background:#f5a623`。active: `transform:translate(2px,2px)` + shadow縮小(押し込み)
- **セカンダリボタン**: `background:#190f06; color:#f5b94a; border:2px solid #8a6428; box-shadow:3px 3px 0 #000`。hover: `border-color:#f5b94a`
- **角丸は一切使わない**(border-radius: 0)
- **見出しの色収差**: `text-shadow: 2px 0 0 rgba(255,60,90,.35), -2px 0 0 rgba(60,180,255,.35)`(大見出しは .4〜.5、起動ロゴは3px)
- 見出しに `◆`、項目ラベルに `▸`、選択カーソルに `▶`、リスト非選択に `・` のプレフィックス
- letter-spacing: 見出し2〜4px、英字ラベル4〜8px

### CRTオーバーレイ(全画面、position:fixed、pointer-events:none)
1. スキャンライン: `repeating-linear-gradient(0deg, rgba(0,0,0,.24) 0 1px, transparent 1px 3px)`
2. ビネット: `radial-gradient(ellipse at center, transparent 52%, rgba(0,0,0,.42) 100%)`
3. フリッカー: `#ffe6b4` を opacity .025〜.06 で `steps(2)` の3.6sアニメ
4. 掃引線: 高さ70pxの薄い明るい帯が7sで上→下へ流れる
設定でオフ可能にすること(prop/フラグ)。

### アニメーション原則
- すべて `steps()` イージング(例: `steps(4)`〜`steps(8)`)でカクカク動かす。滑らかなease禁止
- 画面遷移: `psFadeIn .25s steps(4)`
- 点滅(PRESS START、▼カーソル、サーバー稼働ドット): `psBlink` 1〜1.2s(50%でopacity 0/1切替)

### 効果音(WebAudio、矩形波)
- カーソル/タブ移動: square 560Hz、gain 0.045 → 指数減衰 0.07s
- 決定: 520Hz の後 70ms 遅れて 780Hz
- 戻る/停止: 300Hz、OFF切替: 320Hz
- ヘッダーの「♪ ON/OFF」でミュート可能。AudioContext はユーザー操作後に生成/resume

## ローポリ3Dタンク(ps1-tank.js)
canvas製の回転する醸造タンク。**移植時は `ps1-tank.js` のロジックをそのまま使ってよい**。要点:
- 内部解像度 84×104px の canvas を `image-rendering: pixelated` で拡大(PS1の低解像度感の核)
- ジオメトリ: 回転体。9角形 × [台座スカート / 胴体6リング / 円錐肩 / 煙突 / キャップ]
- 液面: 胴体リング単位で fill%(0-100)以下をアンバー `rgb(235,152,30)`、上をスチール `rgb(128,126,140)`。台座等はダーク `rgb(86,72,56)`
- フラットシェーディング: 面法線×光源の内積を **4段階に量子化**(0.34/0.56/0.78/1.0)
- **頂点スナップ**: 投影後の座標を `Math.round()` で整数化 → PS1特有のポリゴンジッタ
- 描画: painter's algorithm(奥から)、塗り+同色50%暗のストローク1px
- Y軸回転(speed属性、既定0.85rad/s)+ X軸固定チルト0.34rad。煙突上から2×2pxの泡が3つ立ちのぼる
- 属性: `fill`(0-100)、`size`(表示高さpx)、`speed`
- 使用箇所: 起動画面(fill=80/size=170)、タンクカード(fill=進捗%/size=142、黒窓の下端揃え)

## Interactions & Behavior まとめ
- 画面遷移は即時+`psFadeIn .25s steps(4)`。ルーティングは既存の Next.js ページ構成に合わせる(起動画面は初回のみのオーバーレイとして実装可)
- 全クリック操作に効果音(上記仕様)。サウンドトグルで無効化
- Q/Eキー=ワークベンチのタブ巡回
- ボタンは active で2px押し込み+シャドウ縮小
- グリルautoモード: 推奨選択肢を自動回答、質問キューが尽きたら完了状態へ
